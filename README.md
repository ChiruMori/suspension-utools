# 将你的图片悬浮起来的插件

> 就像Setuna那样

![UTOOLS_1604570629229.png](https://i.loli.net/2020/11/05/lYf2L8nEVGZHzrN.png)

选中并按*esc*关闭悬浮窗

[开源代码](https://github.com/qinyongliang/suspension-utools)

快捷键：

- `esc`         关闭窗口
- `方向键`       微调窗口位置
- `+`           放大图片
- `-`           缩小图片
- `[`           向左旋转
- `]`           向右旋转
